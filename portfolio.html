Chattrix: A Messaging App

Chattrix is a real‑time messaging application built with Java and a Swing GUI, designed for devices connected to the same network. The project demonstrates practical implementation of Object‑Oriented Programming (OOP) principles, socket programming, and multithreading to handle multiple client connections simultaneously. The application features a responsive user interface, real‑time message updates, and a simple architecture that makes it efficient for LAN‑based communication.

LANE ESCAPE

Lane Escape is a 2D arcade-style game developed in Java Swing during my second semester, where the player dodges oncoming cars on a multi-lane road. The project demonstrates the use of Object-Oriented Programming by structuring the game into classes for the player, enemy cars, and game logic. Smooth lane-switching controls are implemented through keyboard event handling, while custom sprites and background images create a visually engaging environment. Sound effects and background music enhance the gameplay experience, and collision detection ensures accurate game-over conditions. This project highlights my skills in Java GUI programming, OOP design, and interactive application development.
